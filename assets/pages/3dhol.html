A Gesture-Controlled 3D Hologram System

Project information

  • Category: Mixed Reality - Computer Vision
  • Done for: University Module
  • Project date: 2024
  • Project URL: https://github.com/Daniel2tio/Morbius

This is a project which allows the user to control and manipulate 3D objects from a setting while also being able to use their voice for touch free controls, making it user friendly and natural for all users.

In conclusion, Morbius offers an immersive and engaging experience that seamlessly blends augmented and virtual reality elements, allowing users to interact with holographic content through natural hand gestures. This intuitive interface requires no prior technological expertise, enhancing accessibility and user satisfaction. However, the system's reliance on gesture controls and the slow processing of voice prompts can introduce frustrations, particularly in environments with background noise or for users with motor impairments. The potential for fatigue and cognitive load due to continuous hand use and attentional demands underscores the need for thoughtful design considerations. Enhancing gesture recognition accuracy and incorporating multi-camera setups to improve field-of-view, along with more responsive and inclusive voice controls, can further enhance the user experience, ensuring that Morbius remains both immersive and user-friendly.
